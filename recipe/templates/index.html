Creating a User

To create a new user send a POST request to /users with appropriate JSON data

{{codes.createUser}}

Deleting a User

To delete a user, send a DELETE request to /users/{username}

Only admin or the user can perform this action

{{codes.deleteUser}}

Getting User list

To get a list of users, send a GET request to /users

Use query parameter 'query' to filter users based on name or username

Use query parameter 'count' to specify the number of users returned (max 30)

{{codes.getUserList}}

Getting a specific user

Send a GET request to /users/{username}

{{codes.getUser}}

Getting an AuthToken

To get an auth token, send a GET request to /auth_token

Note: Login credentials or existing auth-token needs to be provided

{{codes.getAuthToken}}

Adding a Recipe

To add a new user send a POST request to /recipes with appropriate JSON data

Note: Login credentials or existing auth-token needs to be provided

{{codes.createRecipe}}

Deleting a Recipe

To delete a recipe, send a DELETE request to /recipe/{recipe-id}

Only admin or the author of the recipe can perform this action

{{codes.deleteRecipe}}
